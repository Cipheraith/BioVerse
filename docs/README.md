# BioVerse Documentation

Welcome to the comprehensive documentation for BioVerse, a health supply chain coordination platform built for African health systems. This directory provides a complete reference for developers, DevOps, and stakeholders.

## Core Philosophy

BioVerse is built on two foundational principles:

1.  **The Unified Platform:** Consolidate fragmented healthcare logistics into a single, intelligent system — from facility-level stock management to national supply chain visibility.
2.  **DHIS2-Native Integration:** We connect directly to DHIS2 (the standard health information system across 80+ countries) to ingest real facility data, compute stock statuses, and generate transfer alerts that prevent stockouts.
